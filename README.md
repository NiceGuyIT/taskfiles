# taskfiles
A collection of task files for use with [task][].

[task]: https://github.com/go-task/task

## Install

Clone this repo.

```bash
git clone https://github.com/NiceGuyIT/taskfiles
cd taskfiles
```

Download [task][].

Initializing the taskfiles downloads the bare minimum of utilities.

```bash
/path/to/task init
```
